    DoDo - Task Management for Hackers

    Dodo is an easily maintainable task list for version controlled projects and hackers. We can call dodo a ticket tracking inside the repo itself. Tasks are called as DoDo, and your goal is to make DoDo's extinct.

    Getting Started

    pip install dodopie

    Initializing

    dodo init
    Options

dodo init -f [dodo filename]

    List all Tasks

    dodo l

    Propose a new Task

    dodo c -u atmb4u -d "dodo new version"
dodo add -u atmb4u -d "dodo new version"
dodo propose -u atmb4u -d "dodo new version"

    Accept a Tasks

    dodo accept --id 2 -u atmb4u -d "dodo new version"

    Reject a proposed Tasks

    dodo reject --id 2 -u atmb4u

    Work on a new Tasks

    dodo workon --id 2 -u atmb4u

    Mark a task as Finished

    dodo finish --id 1 -u atmb4u -d "dodo new version"
